# HIST-2026-020: Bypass de Tenant Header para Super Admin

**Data**: 09/03/2026  
**Status**: ✅ Concluído  
**Tipo**: Correção  
**Tamanho**: Pequeno

## Problema
Endpoints de super-admin (`/api/admin/**`) são operações cross-tenant, mas o filtro de tenant exigia `X-Tenant-ID` e bloqueava chamadas válidas.

## Correção
- Incluído prefixo `/api/admin/` na lista de endpoints sem resolução obrigatória de tenant em `TenantResolverFilter`.

## Critério de aceite
- Chamadas para `/api/admin/**` autenticadas como super-admin não devem falhar por ausência de `X-Tenant-ID`.
